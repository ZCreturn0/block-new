按键订阅：
last-key-update: 订阅最后一次按键
player-move-change: 订阅玩家移动按键
shoot-corner-change: 订阅玩家调整角度按键
shooting-press: 发射球

player-move: 订阅玩家移动事件

碰撞事件：
ball-wall-contacting: 球和墙壁碰撞
ball-player-contacting: 球和玩家碰撞
ball-block-contacting: 球和砖块碰撞

砖块事件：
ball-damage: 球造成伤害

关卡事件：
load-level: 加载关卡
level-clear: 当前关已清除
